This guide covers setting up Frost AI for development and customization. If you just want to get started quickly, check out our [Quick Start](/getting-started/quick-start/) guide.

## Prerequisites

Before installing Frost AI, ensure you have the following installed:

### Required
- **Node.js 18+** - [Download from nodejs.org](https://nodejs.org/)
- **pnpm** - Install with `npm install -g pnpm`
- **Docker & Docker Compose** - [Get Docker](https://docs.docker.com/get-docker/)
- **Git** - [Download Git](https://git-scm.com/)

## Production Installation

### Using Docker Compose (Recommended)

The easiest way to run Frost AI in production is using Docker Compose:

```bash
# Clone the repository
git clone https://github.com/frozen-labs/frost.git
cd frost

# Configure environment for production
cp .env.example .env
# Edit .env with production values

# Start production stack
make prod-up
```

## Database Setup

### Development Database

```bash
# Start PostgreSQL container
make dev-db

# Push schema to database
pnpm db:push

# (Optional) Open database GUI
pnpm db:studio
```

## Verification

After installation, verify everything is working:

### Health Check
```bash
curl http://localhost:3000/api/health
```

Expected response:
```json
{
  "status": "ok",
  "timestamp": "2024-01-01T00:00:00.000Z"
}
```
