The customer management endpoints allow you to create, retrieve, update, and manage customer records in Frost AI. Customers are the top-level entity for organizing usage tracking and billing.

## Endpoints Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/customers` | Create a new customer |
| `GET` | `/api/customers` | List all customers |
| `GET` | `/api/customers/{slug}` | Get a specific customer |
| `PUT` | `/api/customers/{slug}` | Update a customer |
| `DELETE` | `/api/customers/{slug}` | Delete a customer |


## Create Customer

### Endpoint

```http
POST /api/customers
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `slug` | string | ✅ | Unique identifier (URL-safe, lowercase) |
| `name` | string | ✅ | Display name for the customer |

### Request Schema

```json
{
  "slug": "string (1-50 chars, lowercase, alphanumeric + hyphens)",
  "name": "string (1-100 chars)"
}
```

### Response

**Status:** `201 Created`

```json
{
  "id": "01234567-89ab-cdef-0123-456789abcdef",
  "slug": "acme-corp",
  "name": "ACME Corporation",
  "createdAt": "2024-01-01T12:00:00.000Z",
  "updatedAt": "2024-01-01T12:00:00.000Z"
}
```
